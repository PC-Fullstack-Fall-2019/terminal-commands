Commands
--------
* `pwd` - print working directory
* `cd` - change directory
* `mkdir` - make directory
* `ctrl+c` - cancel command
* `rmdir` - remove directory
* `touch` - create file
* `rm` - remove file (-r -f)
* `ls` - list files
* `ls -la` - list all files
* `mv` - move a file

Aliases
-------
* `l` - show all (including hidden)
* `ll` - show all (not hidden)

Paths
-----
* `.` - current directory
* `..` - previous directory
